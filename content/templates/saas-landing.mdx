## Installation through CLI

<Steps>
  <Step>Clone this repository to your local machine:</Step>

```bash
git clone https://github.com/karthikmudunuri/saas-landing-page-template.git
```

  <Step>Move to the cloned directory:</Step>
 
 ```bash
 cd saas-landing-page-template
 ```

  <Step>Install dependencies:</Step>
  ```bash
  npm install
  ```
  
  
  <Step>Start the local Server:</Step>
  ```bash
  npm run dev
  ```

  <Step>Open the [Components](https://github.com/karthikmudunuri/saas-landing-page-template/tree/main/src/components) folder and make changes</Step>



</Steps>
