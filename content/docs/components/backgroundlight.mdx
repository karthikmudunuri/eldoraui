## Props

| Prop               | Type                | Description                                                                                                     | Default                                            |
| ------------------ | ------------------- | --------------------------------------------------------------------------------------------------------------- | -------------------------------------------------- |
| children           | String or Component | The content inside the `BackgroundLight`. This can be any valid JSX such as text, elements, or other components | None                                               |
| showRadialGradient | Boolean             | Determines whether to show a radial gradient mask on the background                                             | true                                               |
| className          | String              | Custom CSS classes to style the outer container of the `BackgroundLight`                                        | ""                                                 |
| initial            | Object              | The initial state for the `motion.div` animation. You can define properties like opacity and position           | `{ opacity: 0.0, y: 40 }`                          |
| whileInView        | Object              | Defines the state the component transitions to when it comes into view (e.g., opacity, position)                | `{ opacity: 1, y: 0 }`                             |
| transition         | Object              | Defines the transition properties (delay, duration, easing) for the `motion.div` animation                      | `{ delay: 0.2, duration: 0.8, ease: "easeInOut" }` |
| id                 | String              | Standard HTML `id` attribute for the container                                                                  | ""                                                 |
| style              | Object              | Inline styles for the container (can be used for custom background colors, etc.)                                | None                                               |
| onClick            | Function            | Event handler for click events on the `BackgroundLight` container                                               | None                                               |
