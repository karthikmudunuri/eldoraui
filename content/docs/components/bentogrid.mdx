### BentoCard Props

| Name          | Type              | Default  | Description                                                             |
| ------------- | ----------------- | -------- | ----------------------------------------------------------------------- |
| `dark`        | `boolean`         | `false`  | Determines if the card should use the dark mode styling.                |
| `className`   | `string`          | `""`     | Additional classes to apply to the root element of the card.            |
| `eyebrow`     | `React.ReactNode` | Required | Content to be displayed as the eyebrow text (usually a subheading).     |
| `title`       | `React.ReactNode` | Required | Content to be displayed as the card title.                              |
| `description` | `React.ReactNode` | Required | Content to be displayed as the card description.                        |
| `graphic`     | `React.ReactNode` | Required | Element to be displayed as the graphic content (e.g., an image or SVG). |
