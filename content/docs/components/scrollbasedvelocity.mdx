## Props

| Prop Name          | Type     | Default | Description                                           |
| ------------------ | -------- | ------- | ----------------------------------------------------- |
| `text`             | `string` | -       | The text to scroll horizontally with velocity effect. |
| `default_velocity` | `number` | `5`     | The default velocity for the scrolling effect.        |
| `className`        | `string` | `""`    | Additional CSS classes for custom styling.            |
