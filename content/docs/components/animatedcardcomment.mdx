## Props

### `CardComment` Component Props

| Prop Name   | Type     | Default | Description                                              |
| ----------- | -------- | ------- | -------------------------------------------------------- |
| `commenter` | `string` | -       | The name of the commenter who made the original comment. |
| `replier`   | `string` | -       | The name of the person who replied to the comment.       |
