## Usage


```tsx showLineNumbers

```

## Props

| Prop                      | Type                       | Default                                                                                                         | Description                                         |
| ------------------------- | -------------------------- | --------------------------------------------------------------------------------------------------------------- | --------------------------------------------------- |
| `id`                      | `string`                   | `'namer-ui-holographic-card'`                                                                                   | Unique identifier for scoped styles                 |
| `width`                   | `number`                   | `320`                                                                                                           | Card width in pixels                                |
| `height`                  | `number`                   | `480`                                                                                                           | Card height in pixels                               |
| `imageSrc`                | `string`                   | `-`                                                                                                             | Main background image URL (required)                |
| `hoverImageSrc`           | `string`                   | `-`                                                                                                             | Optional hover image URL                            |
| `hoverImageEase`          | `string`                   | `"0.3s"`                                                                                                        | Hover image transition duration                     |
| `topText`                 | `React.ReactNode`          | `-`                                                                                                             | Top text overlay content                            |
| `bottomText`              | `React.ReactNode`          | `-`                                                                                                             | Bottom text overlay content                         |
| `mirrorBottomText`        | `boolean`                  | `true`                                                                                                          | Mirror bottom text horizontally                     |
| `isRTL`                   | `boolean`                  | `false`                                                                                                         | Right-to-left text layout                           |
| `textOverlayPadding`      | `string`                   | `"1.375rem 1.325rem"`                                                                                           | Padding for text overlays                           |
| `topTextVertical`         | `boolean`                  | `true`                                                                                                          | Vertical orientation for top text                   |
| `topTextWeight`           | `number \| string`         | `700`                                                                                                           | Top text font weight                                |
| `topTextFontSize`         | `string \| number`         | `"1.5rem"`                                                                                                      | Top text font size                                  |
| `topTextLetterSpacing`    | `string \| number`         | `"0.1em"`                                                                                                       | Top text letter spacing                             |
| `topTextColor`            | `string`                   | `"#fff"`                                                                                                        | Top text color                                      |
| `topTextHoverColor`       | `string`                   | `"#FBE75F"`                                                                                                     | Top text hover color                                |
| `topTextClassName`        | `string`                   | `""`                                                                                                            | Additional CSS classes for top text                 |
| `bottomTextVertical`      | `boolean`                  | `true`                                                                                                          | Vertical orientation for bottom text                |
| `bottomTextWeight`        | `number \| string`         | `700`                                                                                                           | Bottom text font weight                             |
| `bottomTextFontSize`      | `string \| number`         | `"1.5rem"`                                                                                                      | Bottom text font size                               |
| `bottomTextLetterSpacing` | `string \| number`         | `"0.1em"`                                                                                                       | Bottom text letter spacing                          |
| `bottomTextColor`         | `string`                   | `"#fff"`                                                                                                        | Bottom text color                                   |
| `bottomTextHoverColor`    | `string`                   | `"#FBE75F"`                                                                                                     | Bottom text hover color                             |
| `bottomTextClassName`     | `string`                   | `""`                                                                                                            | Additional CSS classes for bottom text              |
| `borderRadius`            | `number`                   | `24`                                                                                                            | Card border radius                                  |
| `borderWidth`             | `number`                   | `1`                                                                                                             | Card border width in pixels                         |
| `borderColor`             | `string`                   | `"rgba(255,255,255,0.1)"`                                                                                       | Card border color                                   |
| `backgroundColor`         | `string`                   | `"#000"`                                                                                                        | Card background color                               |
| `imageOpacity`            | `number`                   | `0.9`                                                                                                           | Background image opacity (0-1)                      |
| `maxImageWidthPct`        | `number`                   | `1`                                                                                                             | Maximum image width as percentage (0-1)             |
| `patternColor`            | `string`                   | `"#000"`                                                                                                        | Noise pattern color                                 |
| `patternOpacity`          | `number`                   | `0.15`                                                                                                          | Noise pattern opacity (0-1)                         |
| `patternSize`             | `number`                   | `3`                                                                                                             | Noise pattern size in pixels                        |
| `patternDotSize`          | `number`                   | `1`                                                                                                             | Noise pattern dot size in pixels                    |
| `enableHologram`          | `boolean`                  | `true`                                                                                                          | Enable hologram overlay effect                      |
| `hologramOpacity`         | `number`                   | `0.4`                                                                                                           | Hologram overlay opacity (0-1)                      |
| `holographicGradient`     | `string`                   | `"linear-gradient(135deg, transparent 35%, rgba(255,0,128,0.4) 45%, rgba(0,255,255,0.4) 55%, transparent 65%)"` | Hologram gradient CSS                               |
| `enableTilt`              | `boolean`                  | `true`                                                                                                          | Enable 3D tilt interaction                          |
| `enableDrag`              | `boolean`                  | `false`                                                                                                         | Enable drag functionality                           |
| `dragConstraints`         | `React.RefObject<Element>` | `-`                                                                                                             | Drag movement constraints                           |
| `className`               | `string`                   | `""`                                                                                                            | Additional CSS classes for container                |
| `style`                   | `React.CSSProperties`      | `-`                                                                                                             | Inline styles for container                         |
