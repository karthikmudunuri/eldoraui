## Usage

```tsx showLineNumbers
import BaubleToggle from "@/components/eldoraui/bauble-toggle"
```

```tsx showLineNumbers
<BaubleToggle
  initialState={true}
  onToggleChange={(isOn) => {
    console.log("Bauble toggle is now:", isOn)
  }}
/>
```

## Props

| Prop           | Type                    | Default | Description                                                |
| -------------- | ----------------------- | ------- | ---------------------------------------------------------- |
| initialState   | boolean                 | true    | Initial on/off value of the bauble toggle.                 |
| onToggleChange | (isOn: boolean) => void | -       | Optional callback fired whenever the toggle state changes. |
