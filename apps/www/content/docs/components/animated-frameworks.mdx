## Usage

```tsx showLineNumbers
import AnimatedFrameworks from "@/components/eldoraui/animated-frameworks"
```

```tsx showLineNumbers
<AnimatedFrameworks
  cardTitle="Universal Compatibility"
  cardDescription="Works seamlessly with Next.js, React, HTML, Apple, GitHub, OpenAI, and more fits everywhere."
/>
```

## Props

| Prop              | Type     | Default                                                                                          | Description                                    |
| ----------------- | -------- | ------------------------------------------------------------------------------------------------ | ---------------------------------------------- |
| `cardTitle`       | `string` | `"Universal Compatibility"`                                                                      | The title displayed at the bottom of the card  |
| `cardDescription` | `string` | `"Works seamlessly with Next.js, React, HTML, Apple, GitHub, OpenAI, and more fits everywhere."` | The description text displayed below the title |
