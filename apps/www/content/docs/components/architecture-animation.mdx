## Usage

```tsx showLineNumbers
import { EldoraUI } from "@/components/eldoraui/architecture-animation"
```

```tsx showLineNumbers
<EldoraUI
  animatedText="npx @eldoraui/cli install"
  appName="Eldoraui"
  circleText="Registry"
  triangleText="AI"
  diamondText="Registry"
  animatedBoxText="Cobe"
  hintText="(components.json)"
/>
```

## Props

| Prop              | Type     | Default                       | Description                                      |
| ----------------- | -------- | ----------------------------- | ------------------------------------------------ |
| `className`       | `string` | `-`                           | Additional CSS classes to apply to the component |
| `circleText`      | `string` | `"Registry"`                  | Text displayed inside the circle shape           |
| `triangleText`    | `string` | `"AI"`                        | Text displayed inside the triangle shape         |
| `diamondText`     | `string` | `"Registry"`                  | Text displayed inside the diamond shape          |
| `animatedText`    | `string` | `"npx @eldoraui/cli install"` | The main animated text with shimmer effect       |
| `animatedBoxText` | `string` | `"Cobe"`                      | Text displayed in the animated box               |
| `hintText`        | `string` | `"(components.json)"`         | Hint text displayed below the main text          |
| `appName`         | `string` | `"Eldoraui"`                  | Application name displayed in the animation      |
