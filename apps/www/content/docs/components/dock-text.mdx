## Usage

```tsx showLineNumbers
import DockText from "@/components/eldoraui/dock-text"
```

```tsx showLineNumbers
<DockText text="Dock Text" />
```

### With Down Prop

```tsx showLineNumbers
<DockText text="Dock Text" down={true} />
```

### Custom Text Size

```tsx showLineNumbers
<DockText text="Dock Text" className="text-4xl" />
```

## Props

| Prop        | Type      | Default | Description                                                                      |
| ----------- | --------- | ------- | -------------------------------------------------------------------------------- |
| `text`      | `string`  | `-`     | The text content to display with dock animation                                  |
| `down`      | `boolean` | `false` | When `true`, the animation scales from the top. When `false`, scales from bottom |
| `className` | `string`  | `-`     | Additional CSS classes to customize the text size and styling                    |
