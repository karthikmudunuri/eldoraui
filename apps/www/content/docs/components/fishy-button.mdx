## Usage

```tsx showLineNumbers
import { FishyButton } from "@/components/eldoraui/fishy-button"
```

```tsx showLineNumbers
<FishyButton
  type="button"
  className="button--1"
  onClick={() => console.log("Ekhad button clicked")}
>
  Ekhad
</FishyButton>
```

```tsx showLineNumbers
<FishyButton
  type="button"
  className="button--2"
  fontFamily="'Roboto Mono', monospace"
  borderRadius="40px"
  width="200px"
  onClick={() => console.log("Roboto Mono clicked")}
>
  Two
</FishyButton>
```

```tsx showLineNumbers
<FishyButton
  type="button"
  className="button--3"
  width="216px"
  fishSpeed="6s"
  fontFamily="'Griffy', cursive"
  borderRadius="50px"
  onClick={() => console.log("Slow fish clicked")}
>
  שלוש
</FishyButton>
```

## Props

| Prop         | Type                          | Default     | Description                          |
|--------------|-------------------------------|-------------|--------------------------------------|
| `children`   | `React.ReactNode`             | `-`         | Button content/text                  |
| `onClick`    | `() => void`                  | `-`         | Click event handler                  |
| `type`       | `"button" \| "submit" \| "reset"` | `"button"` | Button HTML type                   |
| `className`  | `string`                      | `""`        | Additional Tailwind CSS classes      |
| `fontFamily` | `string`                      | `-`         | Custom font family for button text   |
| `borderRadius` | `string`                   | `"20px"`    | Border radius (Tailwind or CSS)      |
| `width`      | `string`                      | `"140px"`   | Button width                         |
| `height`     | `string`                      | `"53px"`    | Button height                        |
| `fishSpeed`  | `string`                      | `"2.3s"`    | Fish swim animation duration         |
