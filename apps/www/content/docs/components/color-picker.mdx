## Usage

```tsx showLineNumbers
import ColorPicker from "@/components/eldoraui/color-picker"
```

```tsx showLineNumbers
<ColorPicker
  value={color1}
  onValueChange={setColor1}
  showAlpha={false}
  showContrast={true}
/>
```

<ColorPicker

## Props

| Prop                            | Type                                | Default                    | Description                                   |
| ------------------------------- | ----------------------------------- | -------------------------- | --------------------------------------------- |
| `value`                         | `string`                            | `"#06B5EF"`                | Current color value (hex format)              |
| `onValueChange`                 | `(val: string) => void`             | `-`                        | Callback when color changes                   |
| `isRTL`                         | `boolean`                           | `false`                    | Enable RTL layout for Hebrew/Arabic           |
| `showContrast`                  | `boolean`                           | `true`                     | Display contrast ratio checker                |
| `enabledModes`                  | `("hex" \| "rgb" \| "hsl")[]`       | `["hex","rgb","hsl"]`      | Available color formats                       |
