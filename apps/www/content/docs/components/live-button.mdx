## Props

| Prop   | Type     | Default | Description                                       |
| ------ | -------- | ------- | ------------------------------------------------- |
| `text` | `string` | `-`     | The text to display on the button                 |
| `url`  | `string` | `-`     | The URL to navigate to when the button is clicked |
