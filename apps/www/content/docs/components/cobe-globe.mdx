## Usage

```tsx showLineNumbers
import { Cobe } from "@/components/eldoraui/cobe-globe"
```

```tsx showLineNumbers
<Cobe variant="default" className="h-[600px] w-full" />
```

### For Variants

```tsx showLineNumbers
<Cobe variant="draggable" className="h-[600px] w-full" />
```

```tsx showLineNumbers
<Cobe variant="auto-draggable" className="h-[600px] w-full" />
```

```tsx showLineNumbers
<Cobe variant="rotate-to-location" className="h-[600px] w-full" />
```

```tsx showLineNumbers
<Cobe variant="scaled" className="h-[600px] w-full" />
```

## Props

| Prop        | Type                                                                               | Default                                         | Description                                     |
| ----------- | ---------------------------------------------------------------------------------- | ----------------------------------------------- | ----------------------------------------------- |
| `variant`   | `"default" \| "draggable" \| "auto-draggable" \| "rotate-to-location" \| "scaled"` | `"default"`                                     | Globe interaction variant                       |
| `className` | `string`                                                                           | `-`                                             | Additional CSS classes for styling              |
| `style`     | `React.CSSProperties`                                                              | `-`                                             | Inline styles for the container                 |
| `locations` | `Location[]`                                                                       | `[{ name: "San Francisco", emoji: "üìç" }, ...]` | Custom locations for rotate-to-location variant |
