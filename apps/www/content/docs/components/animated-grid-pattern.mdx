## Usage

```tsx showLineNumbers

```

```tsx showLineNumbers
<AnimatedGridPattern
  gridSize={8}
  gridWidth={3}
  gridLength={3}
  gap={0.1}
  rippleScale={2}
  rippleRadius={2}
  cornerRadius={0.5}
/>
```

## Props

| Prop             | Type                       | Default                | Description                                      |
| ---------------- | -------------------------- | ---------------------- | ------------------------------------------------ |
| `gridSize`       | `number`                   | `10`                   | Number of grids in each direction (x and z axis) |
| `gridWidth`      | `number`                   | `4`                    | Width of individual grid tiles                   |
| `gridLength`     | `number`                   | `4`                    | Length of individual grid tiles                  |
| `gap`            | `number`                   | `0.05`                 | Spacing between grid tiles                       |
| `rippleScale`    | `number`                   | `2.5`                  | Intensity of the ripple effect on hover          |
| `rippleRadius`   | `number`                   | `2`                    | Radius of the ripple effect from hovered tile    |
| `cornerRadius`   | `number`                   | `0.8`                  | Corner radius of individual grid tiles           |
| `cameraPosition` | `[number, number, number]` | `[-9.31, 12, 24.72]`   | 3D camera position in the scene                  |
| `cameraRotation` | `[number, number, number]` | `[-0.65, -0.2, -0.13]` | 3D camera rotation angles                        |
| `fov`            | `number`                   | `35`                   | Camera field of view                             |
