## Props

| Prop        | Type     | Default | Description                                      |
| ----------- | -------- | ------- | ------------------------------------------------ |
| `text`      | `string` | `""`    | The text content to display with blur-in effect  |
| `className` | `string` | `""`    | Additional CSS classes to apply to the component |
