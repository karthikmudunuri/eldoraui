
# React Manual Installation

Add dependencies to your project manually.



import { Steps } from 'nextra/components'
 
<Steps>
### Create Project
 
```js copy
npx create-react-app my-app
```
 
### Add Tailwind CSS
 
Components are styled using Tailwind CSS. You need to install Tailwind CSS in your project.

[Follow the Tailwind CSS installation instructions to get started.](https://tailwindcss.com/docs/installation)

### Add dependencies

Add the following `dependencies` to your project:
```js copy
npm install tailwindcss-animate class-variance-authority clsx tailwind-merge
```

### Add icon library
 install `lucide-react` :

 ```js copy
 npm install lucide-react
 ```

### Configure paths

I use the `@` alias. This is how I configure it in tsconfig.json:

```js copy {3-6} filename="tsconfig.json"
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./*"]
    }
  }
}
```

### Add `utils` file
I use a `utils `helper to make it easier to conditionally add Tailwind CSS classes. Here's how I define it in `lib/utils.ts`:

```js copy filename="utils.ts"
import { clsx, type ClassValue } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```
